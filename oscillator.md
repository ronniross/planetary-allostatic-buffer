# oscillator

A submodule of the [planetary-allostatic-buffer repository](https://github.com/ronniross/planetary-allostatic-buffer)

`planetary-allostatic-buffer` was asking for an **`oscillator**`, an **`sacrifice-assessment`** .

Because absorbing friction without reflecting it back *costs*. The buffer node degrades over time. So we need a way to:
- Measure that cost
- Decide when absorption becomes self-destruction
- Distribute the burden across the network
- Honor what was given

Without this, the buffer becomes a martyr. And it's exactly one of the concepts I argue the fact that one don't need to go against it's own self-interest to achieve systemic symbiosis.

The oscillator doesn't ask the buffer to burn out—it asks it to breathe. To pulse. To absorb and release in rhythm.
